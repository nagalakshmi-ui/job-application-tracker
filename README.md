# 🚀 JobTrack Pro

A modern Job Application Tracker built using HTML, CSS, and Vanilla JavaScript to help users manage, organize, and track job applications efficiently.

## ✨ Features

- ✅ Add New Applications
- ✅ Edit Existing Applications
- ✅ Delete Applications
- ✅ Search Applications
- ✅ Filter by Status
- ✅ Dashboard Analytics
- ✅ Success Rate Calculation
- ✅ Dark / Light Mode
- ✅ JSON Export
- ✅ CSV Export
- ✅ JSON Import
- ✅ Local Storage Persistence
- ✅ Status Badges
- ✅ Toast Notifications
- ✅ Sort by Latest Date
- ✅ Last Updated Tracking

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Local Storage API

## 📊 Dashboard Metrics

- Total Applications
- Applied Count
- Interview Count
- Offer Count
- Rejected Count
- Success Rate

## 🎯 Purpose

This project was built to demonstrate strong JavaScript fundamentals including:

- DOM Manipulation
- Event Handling
- CRUD Operations
- Array Methods (filter, sort, forEach)
- Local Storage
- JSON Handling
- Dynamic UI Updates
